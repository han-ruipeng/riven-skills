- [ ] 响应式原理（Vue2 defineProperty vs Vue3 Proxy）

- [ ] Fiber 架构
- [ ] 虚拟 DOM & diff
- [ ] 状态管理（Redux / Zustand）

- [ ] Webpack / Vite 原理与配置
- [ ] 性能优化（打包体积、加载速度、运行时性能）
- [ ] 前端监控（错误上报、性能埋点）

### 网络
- [ ] HTTP / HTTPS / HTTP2
- [ ] 跨域及解决方案
- [ ] 浏览器缓存策略

### AI 方向
