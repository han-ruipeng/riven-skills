## 🟢 传出消息 → 主动跟进

### 场景 A：打招呼后 24h 无回复 → 二次触达

```
您好，之前打过招呼，不知道是否方便看一下我的简历？
如果目前不太合适也欢迎直说，感谢！
```

---

### 场景 B：发简历后 48h 无回复 → 轻提醒

```
您好，之前发过简历，想了解一下目前岗位还在招吗？
打扰了，期待您的回复！
```

---

### 场景 C：面试后 3 天无反馈 → 跟进

```
[XX经理/HR]您好，我是[X日]面试[岗位]的[姓名]。
想礼貌跟进一下面试结果，如果有后续安排我随时配合，谢谢！
```

---

### 场景 D：收到 Offer → 缓冲回复

```
非常感谢！请给我[1-2天]时间仔细考虑一下，
我会在[具体日期]之内给您明确答复。
```

---

### 场景 E：已有 offer → 施压其他

```
您好，我这边收到了一个offer需要在XX号之前回复，
想了解一下贵司这边的进度，非常感谢！
```

---

## ❌ 避雷

| ❌ 别这样 | ✅ 应该这样 |
|-----------|------------|
| 连续追问（舔狗感） | 间隔 ≥ 24h 再跟进 |
| "我很需要这份工作" | "我的经验对贵岗匹配度很高" |
| HR 已拒绝还继续纠缠 | 礼貌收尾，留后路 |
| 每条消息都在催回复 | 最多跟进 2 次，第 3 次放弃 |
