# 主动打招呼模板库

## 钩子公式
```
个人亮点(15字) + 匹配理由(25字) + 行动钩子(15字) + 礼貌收尾(10字) = 65字左右
```

---

## 模板 A：数据冲击型（适合有亮眼成果的）

```
您好，我[X]年[岗位方向]经验，主导过[具体项目+量化成果]，
技术栈与贵岗[匹配关键词]高度吻合。
方便看一下我的简历吗？期待回复！
```

### 实例填充（结合 personal-profile.md §二~§四 的个人数据填充）
```
您好，我[X]年[岗位方向]经验，主导过[具体项目+量化成果]，
技术栈与贵岗[匹配关键词]要求高度吻合。
方便看一下我的简历吗？期待回复！
```
> 💡 生成时从 `personal-profile.md` 取技能栈和项目数据填入 `[...]`。

---

## 模板 B：精准匹配型（适合 JD 匹配度高的）

```
您好，看到贵司在招[岗位]，我在[具体领域]有[X]年深耕经验，
[1-2个匹配技能]都是日常主力。
简历已准备好，方便过目吗？
```

---

## 模板 C：转行/跨领域型（适合背景不完全匹配的）

```
您好，虽然我之前主要做[旧领域]，但近[X个月]一直在系统学习[新领域]，
并完成了[具体项目/demo]，希望能有机会详细沟通。
方便聊两句吗？
```

---

## 模板 D：经验丰富型（适合 5 年+）

```
您好，[X]年全栈经验，从[C端/B端/某行业]到[技术方向]都有实际落地，
最近一段在[公司]负责[核心业务]。看到贵岗很感兴趣，方便先看看简历？
```

---

## 避雷指南

| ❌ 不要写 | ✅ 应该写 |
|-----------|----------|
| "我很感兴趣，想聊聊" | "我对贵司XX业务方向感兴趣，因为我有YY经验" |
| "我学习能力强" | "我两周内独立上手了XX框架并交付了YY功能" |
| "求指导求带" | "期待和团队一起解决XX方向的问题" |
| 长篇大论超过100字 | 控制在50-80字，信息密度高 |
| 群发模板腔 | 包含JD中出现的具体关键词 |
