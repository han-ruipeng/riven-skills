# 薪资谈判策略库

## 核心原则

| 原则 | 说明 |
|------|------|
| 让对方先出价 | 先问预算范围，不暴露底牌 |
| 锚定高线 | 期望值定在心理价位 120%，给谈判空间 |
| 谈总包 | base + 年终 + 补贴 + 股票/期权 + 福利 + 年假 |
| 数据支撑 | 引用行业报告、招聘平台数据、上家薪资 |
| 留余地 | "根据整体package综合评估" 而非死咬一个数 |

## 谈判话术

### HR 反问"你期望多少"时的 3 个策略

**策略A：反问法**
```
我更看重平台和发展空间，薪资方面市场有大致区间，
想问一下贵司对这个岗位的预算范围是怎样的？
```

**策略B：锚定法**
```
据我了解贵司这个级别在 [XX-YY] 的范围，
我的期望是 [YY x 1.1]，加上我有 [独有优势]，希望能在这个档位。
```

**策略C：坦白法（有 offer 在手时）**
```
目前我有另一个 offer 在 [数字] 左右，
但如果贵司这边合适，薪资不是唯一考量因素。
```

## 薪资谈判节奏

```
HR 面（初筛薪资范围）
  -> 不提具体数字，确认双方区间有交集即可
技术面（展示价值）
  -> 不谈薪资，专注展示技术实力
终面/HR 面（正式谈判）
  -> 亮出期望值，开始杀价博弈
收到 Offer
  -> 不要当场接受，缓冲1-2天
  -> 如有不满，提出加码诉求
```

## 常见压价话术 & 反制

| HR 压价话术 | 反制策略 |
|-------------|---------|
| "我们预算有限" | "理解，那年终/期权/调薪机制方面有没有弹性空间？" |
| "你经验还年轻" | "虽然年限不长，但我有具体的项目产出数据" |
| "这个行业都这个价" | "我调研的数据显示同岗位中位数是YY，希望能对标" |
| "试用期后再谈" | "可以写进合同，试用期X个月后review薪资调整" |
| "你先来吧，后面涨得快" | "可以在offer里写清楚review机制和时间节点吗？" |

## 不要做的事

- 第一个说出具体数字
- 用"随便"、"都行"回应薪资提问
- 拿"我有房贷/孩子"等个人情况谈价
- 当场接受第一个 offer
- 同时谈多家时透露具体公司名字和 offer 细节
