## 技能列表

### 1. BOSS Job Hunter（BOSS直聘求职助手）`v0.3.0`

求职全流程助手，覆盖职位筛选、主动沟通、面试安排到 offer 复盘的全链路。

| 模块 | 功能 |
|------|------|
| JD 快速筛选 | 根据薪资、地点、行业自动判定职位是否值得投递 |
| 主动沟通 | 生成个性化打招呼消息（数据冲击版 & 技术匹配版） |
| JD 深度解读 | 解析核心需求、必备技能、加分项、潜在坑点 |
| 公司调研 | 评估公司背景、融资阶段、风险评估 |
| 面试安排 | 面试排班管理 + 题库生成 + 模拟面试 |
| 进度跟踪 | 投递/面试状态全记录 |
| 薪资谈判 | HR 面谈策略与话术 |

**触发词**: "帮我在BOSS上找工作"、"帮我分析JD"、"帮我准备面试" 等

---

### 2. Liepin Jobs（猎聘求职工具）`v0.2.0`

原作者 @xllin 修改，做了猎聘官方mcp的更新适配
基于猎聘 MCP Server 的完整求职工具，支持职位搜索、简历投递与在线编辑。

| 功能 | 说明 |
|------|------|
| 职位搜索 | 按名称、地点、薪资、经验、学历、公司性质等筛选 |
| 简历投递 | 一键投递（需用户确认） |
| 简历管理 | 查看/编辑基本信息、教育经历、工作经历、项目经历、求职意向、自我评价 |

**依赖**: Python 3 + 猎聘账号 Token（首次使用需配置）

**触发词**: "找工作"、"搜职位"、"投简历"、"猎聘" 等

### 3. File Organizer（文件整理工具）`v1.0.0`

将指定文件夹中散乱的文件按扩展名自动归类到子文件夹。

| 类别 | 目标文件夹 |
|------|-----------|
| 图片 | `images/` |
| 文档 | `documents/` |
| 压缩包 | `archives/` |
| 代码 | `code/` |
| 视频 | `videos/` |
| 音频 | `audio/` |
| 安装包 | `installers/` |
| 其他 | `others/` |

**触发词**: "整理文件夹"、"分类文件"、"清理下载" 等
